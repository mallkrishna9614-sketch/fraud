# AI Fraud Detection System

A modern Next.js dashboard for real-time fraud monitoring with interactive graph exploration, explainable risk context, live transaction activity, and API-ready data loading.

## Stack
- Next.js + React + TypeScript
- Tailwind CSS
- Framer Motion
- React Flow
- Recharts

## API placeholders
The frontend is ready to consume these endpoints:
- `GET /transactions`
- `GET /fraud-users`
- `GET /graph-data`

If an endpoint is unavailable, mock dynamic data generation is used as fallback so the dashboard remains functional.

## Run locally
```bash
npm install
npm run dev
```
